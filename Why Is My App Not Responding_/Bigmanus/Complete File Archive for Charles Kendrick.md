## USAGE INSTRUCTIONS

### To Deploy Lecture Me:

1. **Extract Files:**
   ```bash
   tar -xzf 05_ALL_SOURCE_CODE.tar.gz
   cd lecture-me-pro
   ```

2. **Install Dependencies:**
   ```bash
   pnpm install
   ```

3. **Configure Environment:**
   - Copy `.env.example` to `.env`
   - Add database URL
   - Add API keys (Stripe, OAuth, etc.)

4. **Run Development Server:**
   ```bash
   pnpm dev
   ```

5. **Deploy to Production:**
   - Use Manus platform (current)
   - Or deploy to Vercel, AWS, Railway, etc.
   - Point lectureme.org to new host
